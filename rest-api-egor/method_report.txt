Метод парных сравнений
Метод предусматривает использование эксперта, который проводит
оценку целей:
Z1, Z2, ...,Zn.
Согласно методу осуществляются парные сравнения целей во всех
возможных сочетаниях. В каждой паре выделяется наиболее
предпочтительная цель. И это предпочтение выражается с помощью оценки
по какой-либо шкале. Обработка матрицы оценок позволяет найти веса
целей, характеризующие их относительную важность. Одна из возможных
модификаций метода состоит в следующем:
1) составляется матрица бинарных предпочтений, в которой
предпочтение целей выражается с помощью булевых переменных;
2) определяется цена каждой цели путем суммирования булевых
переменных по соответствующей строке матрицы.
